What JrBTC is

JrBTC is not a token on someone else's chain. It is its own Layer-1 network — the Junior Network — built by forking Bitcoin Core v31.1 and changing only the chain parameters. Every JrBTC is created by miners running the node software and securing the network, exactly as Bitcoin works.

  • Real Bitcoin engine. libsecp256k1 signatures, double-SHA-256 proof-of-work, UTXO ledger, Bitcoin Script with SegWit & Taproot, and the Bitcoin P2P protocol — inherited unchanged.
  • Honest, disclosed supply. A one-time 2,000,000 JrBTC Founder Reserve is paid at block 1 by a consensus rule stated in the whitepaper and visible on-chain forever. Every other coin — roughly 19 million — can only be mined.
  • Legible. The entire divergence from Bitcoin Core is four files and ~77 lines, published as a patch anyone can review.

How to mine JrBTC

Mining happens on your own computer by running the node software — not in a browser and not on this website. Here is the whole flow:

  1. Get the software. Download the prebuilt Windows kit from GitHub Releases, or build from source for any OS (build guide).
  2. Start your node and point it at a running Junior Network peer (a "seed node"): 
    start-node.cmd <seed-ip>
    Your node downloads the chain and verifies every block back to genesis itself.
  3. Mine to your wallet. Each block pays 50 JrBTC to an address you control (spendable after 100 confirmations): 
    mine.cmd
    Advanced miners can point standard SHA-256 mining software at the node's getblocktemplate RPC, just like Bitcoin.

Want to host the network's seed node?

A live network needs at least one always-on node with P2P port 9333 reachable — a small VPS, or a home PC with the port forwarded. Share its address and anyone can join and mine against it. Full instructions are in the repo's README.

Read this before you mine or value it

Honest disclaimers

  • This is a brand-new, experimental network. Its security depends on how much honest mining power it has, which starts small. Treat it as a technical project to learn on and build with — not as a store of value.
  • Not investment advice, and no promise of profit or future price. JrBTC is not sold; coins are mined. Nothing here is an offer of securities.
  • You cannot add JrBTC to Coinbase, Trust Wallet, or MetaMask. It is its own Layer-1; only Junior Network node software holds it. Never import your private keys into a third-party app hoping it will "find" your coins.
  • Back up your wallet file. Whoever holds it holds the coins. Lose it and they are gone — the same rule Bitcoin has.
  • The Founder Reserve is disclosed, not hidden — 2M of the 21M, on-chain at block 1, for liquidity and grants. That is roughly 9.5% of the cap; decide for yourself what that means to you.
